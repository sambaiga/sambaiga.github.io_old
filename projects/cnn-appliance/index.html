CNN for Appliance Recognition in Energy Disaggregation (NILM)

Appliance recognition is an important sub-task of the NILM problem. It involve detecting which and when an appliance change state. Several approaches using signal processing and machine learning techniques have been proposed for this sub-task. In most of these approaches deep-learning have received little attention.

This work complement the work by Gao, Jingkun, et al with convolutional neural network (CNN) using Plug load Appliance Identification Dataset (PLAID).
